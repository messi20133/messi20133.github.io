给定数组a[N]构造数组b [N]


题目：给定一个数组a[N]，我们希望构造数组b [N]，其中b[j]=a[0]*a[1]…a[N-1] / a[j]，在构造过程中，不允许使用除法：要求O（1）空间复杂度和O（n）的时间复杂度；除遍历计数器与a[N] b[N]外，不可使用新的变量（包括栈临时变量、堆空间和全局静态变量等）

分析：
设 b[0]=1;
令b[i]=b[i-1]a[i-1]
所以：
b[1]=a[0]
b[2]=b[1]a[1]=a[0]a[1];
.
.
.
b[i]=a[0]*….a[i-1]
通过b[0]这个变量来迭代出1, a[n-1], a[n-2]a[n-1], a[n-3]a[n-2]a[n-1], … , a[1]a[2]a[3]…a[n-1]，迭代过程中分别乘以b[n-1], b[n-2], … , b[0]

代码如下：

package StringAndArray;

import java.util.Arrays;

public class CreareArrayB {
public static void Translate(int A[],int B[]){
    int n=A.length;
    B[0]=1;
    for(int i=1;i<n;i++)
        B[i]=B[i-1]*A[i-1];
    for(int i=n-1;i>=1;i--){
        B[i]=B[i]*B[0];
        B[0]=B[0]*A[i];
    }
}
public static void main(String[] args) {
    int[] A={2,3,4,5};
    int[] B=new int[4];
    Translate(A,B);
    System.out.println(Arrays.toString(A));
    System.out.println(Arrays.toString(B));
}
}
