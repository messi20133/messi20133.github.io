与链表相关的操作


下面介绍一下与链表有关的操作：
1.翻转链表，即a->b->c->d 变成d->c->b->a

public static Node reverse_link(Node head){
    if(head==null)
        return null;
    Node pre=null,curr=null,reverse_head=null,temp;
    curr=head;
    while(curr.next!=null){
        temp=curr.next;
        curr.next=pre;
        pre=curr;
        curr=temp;
    }
    curr.next=pre;
    reverse_head=curr;
    return reverse_head;

}

2.递归方式实现链表的翻转

public static Node reverse_link_recursive(Node head){
    if(head==null)
        return null;
    Node temp,curr,reverse_head;
    if(head.next==null)
        return head;
    else{
        curr=head;
        temp=curr.next;
        reverse_head=reverse_link_recursive(temp);
        temp.next=curr;
        curr.next=null;

    }
    return reverse_head;
}

3.链表相邻元素翻转如a->b->c->d->e->f-g，翻转后变为：b->a->d->c->f->e->g

public static Node reverseLinkList3(Node head){
    if(head==null)
        return null;
    Node pre=null;
    Node cur=head;
    Node next=null;
    while(cur!=null&&cur.next!=null){
        if(pre==null){
            head=cur.next;
        }else{
            pre.next=cur.next;
        }
        pre=cur;
        next=cur.next.next;
        cur.next.next=cur;
        cur.next=next;
        cur=next;
    }
    return head;
}
