将一个整数nSum分解成num个整数和的形式，如nSum=6，num=3那么，nSum就可以分解成为1 1 4；1 2 3；222。


如题：将一个整数nSum分解成num个整数和的形式，如nSum=6，num=3那么，nSum就可以分解成为1 1 4；1 2 3；222。请编程实现！

分析：采用递归的算法，分解时，层次（nDepth，基于0）每深一层，其层所对应的值就应该不比上一层的值小，所以循环体中变量i应从上一层的值开始，直到nSum/num时截止，因为最后一组最大情况应都为nSum/num。其中nDepth+num的值为总的分解个数。

代码如下：

package PermutationsAndcombinations;
/*
 * 将一个整数nSum分解成num个整数和的形式，如nSum=6，num=3那么，nSum就        可以分解成为1 1 4；1 2 3；222。
 */
public class SpiteNum {
public static void SegNum(int nSum,int num,int pData[],int pDepth){
    if(1==num){
        //当num=1的时候输出pData中的值，
        pData[pDepth]=nSum;  //别忘了把nSum加上
        for(int j=0;j<num+pDepth;j++){
            System.out.print(pData[j]+" ");
        }
        System.out.println();
    }else{
        int i=(pDepth==0?1:pData[pDepth-1]);
        for(;i<=nSum/num;i++){
            pData[pDepth++]=i; 
            SegNum(nSum-i, num-1, pData, pDepth); //开始深的一层
            pDepth--;
        }
    }
}
public static void main(String[] args) {
    //测试代码
    int nSum=10;
    int num=3;
    int[] pData=new int[num];
    SegNum(nSum,num,pData,0);
    }
}
