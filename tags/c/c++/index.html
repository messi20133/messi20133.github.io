const学习

const修饰指针的情况一般分为如下四种：

int  b=500;  
const int * a=&b;  //情况1
int const *a=&b;  //情况2
int * const a=&b;  //情况3
const int * const a=&b;  //情况4

当const位于星号左侧，则const就是用来修饰指针所指向的变量，即，指针指向的为常量，
当const位于星号右侧，则const就是用来修饰指针本身，即，指针本身为常量

对于情况1

int b=500;
const int *a=&b;
*a=600;//错误

也有别的办法修改*a的值，一种方法是通过改变b的值来改变a的值

int b=500;
const int *a=&b;
b=600;

另一种办法就是将a指向别处

int b=500，c=600;
const int *a=&b;
a=&c;

对于情况1，可以先不进行初始化，因为虽然指针的内容是常量，但是指针不是常量。
对于情况3，必须同时初始化

int b=500,c=500;
int * const a;//错误，没有初始化
int * const a=&b //正确
*a=600; //正确，允许修改值
a=&c;//错误，

对于情况4，指针本身和指针指向的内容均为常量。

const成员函数:
只读函数，不能修改数据成员的值。

const与#define相比有什么不同？
const常量有数据类型，而宏常量没有数据类型，编译器对前者进行类型安全检查，对后者只是进行字符替换，没有安全性检查。

如何在const函数中修改数据成员变量？
在数据成员前面加上mutable
